# ChatBoat 🚢

A real-time room-based chat app built with **React + Tailwind CSS**.

## Features
- 🌐 **Room-based chat** — IP address as the room code
- ↩️ **WhatsApp-style reply** — Hover over any message to reply
- 😊 **Emoji Picker** — Full emoji keyboard
- ❤️ **Message Reactions** — 👍 ❤️ 😂 😮 😢 🔥 reactions
- 🖼️ **Image Sharing** — Share images up to 5MB
- ✍️ **Typing Indicator** — Animated dots when someone is typing
- 👥 **Online Members Panel** — See who's in the room
- 🗑️ **Clear Chat** — Delete all messages with confirmation
- 📱 **Fully Responsive** — Mobile + Desktop
- ☀️ **Clean Light UI** — White + Brand Blue theme

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000)

## Multi-Tab Testing
Open the same room in multiple browser tabs to test the chat.

## Real Multi-Device
Replace `useChat.js` hook with Socket.io or Firebase for real networking.
